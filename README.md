# Cancer Incidence Analysis

Code accompanying:

**Immune Decline, Mutation Accumulation and the Structure of Cancer Incidence**  
Zahid et al.

## Reproducing the manuscript

Run notebooks in order:

1. Figure1.ipynb
2. Figure2.ipynb
3. Figure3.ipynb
4. Figure4.ipynb
5. Figure5.ipynb

Figure 4 reproduces the cancer-incidence model fits and generates the parameter files used by Figure 5.
